

# BadgerMint Landing Page

A modern, high-converting landing page for BadgerMint — the smart event and networking platform.

## Design Direction
- Clean, premium tech/startup aesthetic with subtle gradients (mint greens, deep navy/charcoal accents)
- Smooth fade-in and scroll animations throughout
- Mobile-first, fully responsive layout
- Light mode design (dark mode friendly color system)

## Page Sections

### 1. Navigation Bar
- BadgerMint logo/wordmark on the left
- Links: Features, How It Works, Use Cases, For Developers
- CTA buttons: "Join an Event" and "Get Started"
- Sticky header with blur backdrop on scroll

### 2. Hero Section
- Bold headline emphasizing the core value: "One Tap. Infinite Opportunities."
- Subheadline explaining the smart card + event platform concept
- Two CTAs: "Get Started" (primary) and "See How It Works" (secondary)
- Animated illustration/mockup area showing the tap-to-connect flow

### 3. How It Works
- 3-step visual flow: **Tap Card → Share Profile → Connect Instantly**
- Each step as an animated card with icon and short description
- Clean connecting line or arrow between steps

### 4. Features Grid
- Card-based layout showcasing: Smart Card Sharing, Digital Resume/Portfolio, Live Event Interaction, Micro Apps, App Marketplace
- Each card with an icon, title, and 1-2 line description
- Subtle hover animations

### 5. Use Cases
- Tab or card layout for: Events & Conferences, Job Fairs, Business Networking, Audience Engagement
- Each use case with a brief scenario description and relevant icon

### 6. Micro Apps & Extensions Showcase
- Visual grid of sample micro apps: Live Polls, Q&A, Voting, Giveaways, Check-ins, Networking Tools
- App-store-style cards with icons and short labels

### 7. For App Developers
- Section targeting third-party developers to build and add micro apps
- Highlights: open platform, reach event audiences, easy integration
- CTA: "Build on BadgerMint" or "Partner With Us"

### 8. Final Call-to-Action
- Full-width section with bold messaging
- Three CTA paths: "Get Started", "Join an Event", "Partner With Us"

### 9. Footer
- Logo, navigation links, social media icons
- Links: About, Privacy, Terms, Contact
- Clean, minimal layout

## Interactions & Polish
- Fade-in animations on scroll for each section
- Hover effects on cards and buttons
- Smooth scroll navigation from the navbar
- Responsive: stacked layouts on mobile, multi-column on desktop
